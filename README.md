# D'Cave Nightclub Cancún - Landing Page

Landing page para D'Cave Nightclub Cancún con múltiples propuestas de diseño.

## Estructura del Proyecto

### Archivos Principales

- `index.html` - Versión principal de la landing
- `index-propuesta1.html` - Propuesta 1: Sección dedicada de tickets con tabs y grid de cards
- `index-propuesta2.html` - Propuesta 2: Sección de booking integrada
- `index-propuesta3.html` - Propuesta 3: Modal/popup para productos

### Carpetas

- `css/` - Archivos de estilos CSS
- `js/` - Archivos JavaScript
- `images/` - Imágenes y assets visuales
- `fonts/` - Fuentes tipográficas
- `media/` - Videos y contenido multimedia
- `Fotos y videos/` - Material adicional de fotos y videos

## Características

### Propuesta 1
- Sección dedicada `#pricing` con tabs (Table Service, Personal, Residents, Special Offers)
- Grid de cards para productos
- Botones del header y hero apuntan a `#pricing`

### Propuesta 2
- Sección de booking `#booking` integrada
- Botones del header y hero apuntan a `#booking`

### Propuesta 3
- Modal/popup para mostrar productos
- Botones del header y hero abren el modal
- Tabs dentro del modal para diferentes categorías

## Funcionalidades

- **Scroll suave** entre secciones
- **Header sticky** con efecto headroom
- **Responsive design** para desktop, tablet y móvil
- **Galería de imágenes** con lightbox
- **Mapa de ubicación** integrado
- **Integración con MandalaTickets** para reservas

## Tecnologías Utilizadas

- HTML5
- CSS3 (con animaciones y transiciones)
- JavaScript (vanilla)
- Bootstrap (para grid y componentes)
- jQuery (para algunas funcionalidades)
- Font Awesome (iconos)

## Navegación

Los botones "BOOK TICKETS" y "BOOK YOUR TICKETS AND VIP" están configurados para:
- **Propuesta 1**: Llevar a la sección `#pricing`
- **Propuesta 2**: Llevar a la sección `#booking`
- **Propuesta 3**: Abrir el modal de productos

## Notas de Desarrollo

- El scroll suave está habilitado en todas las propuestas
- Los estilos están inline en algunos casos para facilitar la portabilidad
- Las imágenes y videos están optimizados para web
- Compatible con navegadores modernos (Chrome, Firefox, Safari, Edge)
