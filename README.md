# D'Cave Nightclub Cancún - Landing Page

Landing page para D'Cave Nightclub Cancún con múltiples propuestas de diseño.

## Estructura del Proyecto

### Archivos Principales

- `index.html` - Versión principal de la landing

**Versiones con fondo negro:**
- `index-propuesta1.html` - Propuesta 1: Sección dedicada de tickets con tabs y grid de cards (fondo negro)
- `index-propuesta2.html` - Propuesta 2: Sección de booking integrada (fondo negro)
- `index-propuesta3.html` - Propuesta 3: Modal/popup para productos (fondo negro)

**Versiones con fondo gris oscuro:**
- `index-propuesta1-gris.html` - Propuesta 1: Sección dedicada de tickets con tabs y grid de cards (fondo gris oscuro #0f0f0f)
- `index-propuesta2-gris.html` - Propuesta 2: Sección de booking integrada (fondo gris oscuro #0f0f0f)
- `index-propuesta3-gris.html` - Propuesta 3: Modal/popup para productos (fondo gris oscuro #0f0f0f)

### Carpetas

- `css/` - Archivos de estilos CSS
- `js/` - Archivos JavaScript
- `images/` - Imágenes y assets visuales
- `fonts/` - Fuentes tipográficas
- `media/` - Videos y contenido multimedia
- `Fotos y videos/` - Material adicional de fotos y videos

## Características

### Propuesta 1 (Negro y Gris Oscuro)
- Sección dedicada `#pricing` con tabs (Table Service, Personal, Residents, Special Offers)
- Grid de cards para productos
- Botones del header y hero apuntan a `#pricing`
- Disponible en dos variantes: fondo negro (#000000) y fondo gris oscuro (#0f0f0f)

### Propuesta 2 (Negro y Gris Oscuro)
- Sección de booking `#booking` integrada
- Botones del header y hero apuntan a `#booking`
- Disponible en dos variantes: fondo negro (#000000) y fondo gris oscuro (#0f0f0f)

### Propuesta 3 (Negro y Gris Oscuro)
- Modal/popup para mostrar productos
- Botones del header y hero abren el modal
- Tabs dentro del modal para diferentes categorías
- Disponible en dos variantes: fondo negro (#000000) y fondo gris oscuro (#0f0f0f)

## Variantes de Color

El proyecto incluye 7 versiones en total:
1. `index.html` - Versión original
2. `index-propuesta1.html` - Propuesta 1 con fondo negro
3. `index-propuesta2.html` - Propuesta 2 con fondo negro
4. `index-propuesta3.html` - Propuesta 3 con fondo negro
5. `index-propuesta1-gris.html` - Propuesta 1 con fondo gris oscuro (#0f0f0f)
6. `index-propuesta2-gris.html` - Propuesta 2 con fondo gris oscuro (#0f0f0f)
7. `index-propuesta3-gris.html` - Propuesta 3 con fondo gris oscuro (#0f0f0f)

**Nota sobre el gris oscuro:** El color #0f0f0f fue seleccionado por ser un gris oscuro profesional que mejora la legibilidad y el contraste, optimizado para aumentar conversiones y ventas.

## Funcionalidades

- **Scroll suave** entre secciones
- **Header sticky** con efecto headroom
- **Responsive design** para desktop, tablet y móvil
- **Galería de imágenes** con lightbox
- **Mapa de ubicación** integrado
- **Integración con MandalaTickets** para reservas

## Tecnologías Utilizadas

- HTML5
- CSS3 (con animaciones y transiciones)
- JavaScript (vanilla)
- Bootstrap (para grid y componentes)
- jQuery (para algunas funcionalidades)
- Font Awesome (iconos)

## Navegación

Los botones "BOOK TICKETS" y "BOOK YOUR TICKETS AND VIP" están configurados para:
- **Propuesta 1** (negro y gris): Llevar a la sección `#pricing`
- **Propuesta 2** (negro y gris): Llevar a la sección `#booking`
- **Propuesta 3** (negro y gris): Abrir el modal de productos

## Notas de Desarrollo

- El scroll suave está habilitado en todas las propuestas
- Los estilos están inline en algunos casos para facilitar la portabilidad
- Las imágenes y videos están optimizados para web
- Compatible con navegadores modernos (Chrome, Firefox, Safari, Edge)
